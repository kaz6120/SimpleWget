Requirements

Mac OS X v.10.5 or later.
Mac OS X v.10.5以降が必要です。

How to use

Set URI. Choose Download Folder. Choose your options and do "GET".
You can also load the list of URLs from a text file.
Check out the popup help of each options for details. Have Fun!

URIをセットし、ダウンロードフォルダを選択します。
Wgetオプションを選択し、GETボタンを押すとダウンロードが開始されます。
オプションの詳細はポップアップヘルプを参照下さい。
